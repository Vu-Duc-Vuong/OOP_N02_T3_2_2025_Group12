# Spring Boot GUI - Quản Lý Hàng Hóa

## 🚀 Quick Start

### Cách chạy nhanh:
```bash
cd gs-serving-web-content-main/complete
chmod +x mvnw
./mvnw spring-boot:run
```

### Truy cập ứng dụng:
- URL: http://localhost:8080
- Quản lý hàng hóa: http://localhost:8080/hanghoa

## 📖 Hướng dẫn chi tiết
